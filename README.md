# Email → Ticket Agent (Outlook-first)

Phase 0 scaffold for the Email→Ticket project.

## What this repo contains (Phase 0)
- project scaffold for watcher, parser, web, db
- dev environment setup instructions
- pre-commit hooks (black, isort, flake8)
- a placeholder unit test

## Quick start
1. Create a virtualenv:
   - `python -m venv .venv`
   - mac/linux: `source .venv/bin/activate`
   - windows: `.venv\Scripts\activate`
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Run tests:
   - `pytest`

See `.env.example` for environment variables to configure later.
